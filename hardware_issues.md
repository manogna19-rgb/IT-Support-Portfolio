# Hardware Issues (IT Support Practice)

## Scenario 1: Printer Not Working

Issue:
User reports printer is not working

Steps:
- Check power and cable connections
- Restart printer
- Check printer queue for stuck jobs
- Reinstall or update printer drivers if needed

Resolution:
Printer started working after clearing queue/restart

---

## Scenario 2: Laptop Not Booting

Issue:
Laptop is not turning on or not booting properly

Steps:
- Check power adapter and battery
- Try different power outlet
- Perform hard reset
- Check for indicator lights or sounds
- Escalate if hardware issue suspected

Resolution:
Issue identified or escalated for repair

---

## Scenario 3: External Monitor Not Detected

Issue:
External monitor not displaying

Steps:
- Check cable connections (HDMI/VGA)
- Try different port or cable
- Detect display in system settings
- Update display drivers

Resolution:
Monitor successfully connected and detected
