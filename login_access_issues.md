Login & Access Issues (IT Support Practice)

## Scenario 1: Password Reset

Issue:
User cannot login due to incorrect password

Steps:
- Verify username and check caps lock
- Confirm identity using OTP or security questions
- Reset password
- Ask user to login again

Resolution:
User successfully logged in

---

## Scenario 2: Account Lockout

Issue:
User account is locked after multiple failed login attempts

Steps:
- Check account status in system
- Unlock the account
- Inform user about password attempts
- Ask user to login again

Resolution:
User regained access successfully

---

## Scenario 3: MFA (Multi-Factor Authentication) Issue

Issue:
User is not receiving OTP or cannot complete MFA login

Steps:
- Check if the user has network connectivity
- Verify registered phone number or email
- Ask user to retry receiving the code
- Resend or reconfigure MFA settings if needed
- Escalate if issue persists

Resolution:
User successfully completed authentication
