# Network Issues (IT Support Practice)

## Scenario 1: No Internet

Issue:
User reports no internet connection

Steps:
- Check if other devices are affected
- Restart router/modem
- Check Wi-Fi or Ethernet connection
- Run basic commands (ipconfig, ping)

Resolution:
Internet connection restored

---

## Scenario 2: Slow Internet

Issue:
User reports slow internet speed

Steps:
- Check bandwidth usage
- Close background applications
- Restart system and router
- Test internet speed

Resolution:
Performance improved after optimization

---

## Scenario 3: Cannot Access Shared Drive

Issue:
User unable to access shared network drive

Steps:
- Check network connectivity
- Verify user permissions
- Reconnect mapped drive
- Restart system if needed

Resolution:
User regained access to shared resources
